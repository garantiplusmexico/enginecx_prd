# Feature Nuevo rol Distribuidor-Taller

Se requiere habilitar un nuevo rol que se llamará Usuario Distribuidor-Taller
Este usuario deberá contar con las caracteristicas que tiene actualmente el rol Usuario Distribuidor y Usuario Taller

en el catalogo de usuarios, al crear este tipo de usuario (Usuario Distribuidor-Taller) se debe permitir seleccionar
el o los distribuidores que estarán relacionados al usuario, asi como el taller al que estará relacionado el usuario.

Las opciones permitidas para este usuario deberá corresponder las permitidas a los usuarios Usuario Distribuidor y Usuario Taller.
